# Cheers!!

## サイト概要
情報共有サイト

### サイトテーマ
自宅でお酒を楽しむために有益な情報を共有する

### テーマを選んだ理由
* 自分自身、お酒が好き
* コロナ禍で家で飲む機会が増えた  
この２点から、みんなはどう嗜んでいるのか気になりました。お酒の種類、おつまみ、何をしながらなのか、いろんな情報を共有して皆さんの宅飲みをより楽しいものにできたらと考えたため

### ターゲットユーザ
お酒を嗜む方

### 主な利用シーン
* 家で飲んでいて何か物足りなくなった時
* 家で飲む機会が増えた時
